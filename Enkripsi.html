Text Encryption/Decryption Tool

How to use:

1. Enter your text in the top box

2. Set an encryption key (keep it secret!)

3. Choose a shift value (1-100)

4. Click "Encrypt" to encrypt your text

5. To decrypt, paste the encrypted text and click "Decrypt" using the same key and shift value

Note: You must use the same key and shift value to decrypt successfully.
